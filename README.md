## 概要
* このゲームは複数チームに分かれて、敵のコアを削り、ゼロにした方が勝ちというゲームです（某ゲームと近いです）。

## コマンド
* /game_start ：ゲーム開始するコマンド
* /game_reset ： ゲームをリセットするコマンド
